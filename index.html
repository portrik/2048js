About 2048JS

This game is reimagination of the classic 2048 using pure JavaScript and HTML Canvas element. It was created as a semestral project at FEE CUT.

Arrow keys, WASD and touchscreens are all supported as input methods.

If you would like to know more about the game or are interested in the source code, visit the Github repo.

The mobile menu icon was created by Icons8.
